How to build : use command "make" in the folder.
How to execute : use command "./Lab3 [input].aux"
	for example : "./Lab3 newblue5.aux"

The output file will be named : output.pl

And the follow excution for example :
	./Lab3_verifier newblue5.aux output.pl
	./Lab3_evaluate newblue5.pl output.pl

===================================================================
The bigger case will take some time (newblue5 : 300s), plaese be patient.
